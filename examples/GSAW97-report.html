Austrian national guest survey

This document reproduces the example “Austrian national guest survey” presented in the paper “Probabilistic archetypal analysis” by Seth and Eugster (2014). The Rmd file lists the corresponding R code.

  • Read tourist data from file GSAW97_bin.csv
  • Load precomputed PAA solution from file GSAW97_paa.mat
