Disasters worldwide from 1900-2008: AA versus PAA

This document reproduces the comparison of the probabilistic and the classical archetypes in the example “Disasters worldwide from 1900-2008” presented in the paper “Probabilistic archetypal analysis” by Seth and Eugster (2014). The Rmd file lists the corresponding R code.

  • Read disaster data from file emdata.csv
  • Load precomputed PAA solution from file Disaster_paa.mat
  • Load precomputed AA solution from file Disaster_aa.Rds
  • Match AA solution with PAA solution
